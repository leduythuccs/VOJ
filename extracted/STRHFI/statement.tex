   Với 1 số   \textbf{    d   }   cho trước, bạn được cho   \textbf{    N   }   -   \textbf{    d   }   +1 xâu con s[1]-$>$s[   \textbf{    d   }   ], s[2]-$>$ s[   \textbf{    d   }   +1]... s[   \textbf{    N   }   -   \textbf{    d   }   +1]-$>$s[   \textbf{    N   }   ]. Hãy tìm 2 kí tự đầu và cuối của s.
